## Descrição
API simples com observabilidade via Prometheus e visualização de métricas no Grafana.

## Endpoints
- `GET /health`
- `GET /books`
- `POST /books`
- `GET /metrics`

## Executar o Projeto

```bash
docker-compose up --build
```

## Acessos
- API: http://localhost:5000
- Prometheus: http://localhost:9090
- Grafana: http://localhost:3000
